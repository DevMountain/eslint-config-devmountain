# eslint-config-devmountain

## Usage:

`npm i eslint-config-devmountain --save-dev`

Add one of the following lines to your `.eslintrc`:

### ES5

```
{
	"extends": "devmountain" 
}
```

### ES6

```
{
	"extends": "devmountain/es6-config" 
}
```

### React

```
{
	"extends": "devmountain/react-config" 
}
```
